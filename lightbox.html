How to Use

1. Include the CSS and JS

<link rel="stylesheet" href="./css/styles.css">
<script src="./js/components/lightbox.js"></script>

2. Add the Lightbox HTML to your page (before closing </body>)

<!-- Lightbox Component -->
<div id="lightbox" class="lightbox" aria-hidden="true">
    <div class="lightbox-backdrop"></div>
    <div class="lightbox-content">
        <button class="lightbox-close" aria-label="Close">...</button>
        <div class="lightbox-media">
            <img class="lightbox-image" src="" alt="" style="display: none;">
            <iframe class="lightbox-pdf" src="" style="display: none;"></iframe>
        </div>
        <button class="lightbox-prev" aria-label="Previous">...</button>
        <button class="lightbox-next" aria-label="Next">...</button>
        <div class="lightbox-counter"></div>
    </div>
    <div class="lightbox-thumbnails"></div>
</div>
<script src="./js/components/lightbox.js"></script>

Note: SVG icons omitted for brevity. See full HTML at the bottom of this page.

3. For Image Galleries

<img src="image1.jpg" alt="Image 1" data-lightbox="gallery-name" data-index="0">
<img src="image2.jpg" alt="Image 2" data-lightbox="gallery-name" data-index="1">
<img src="image3.jpg" alt="Image 3" data-lightbox="gallery-name" data-index="2">

4. Trigger Gallery from Any Element

<button
    data-lightbox-trigger="team-gallery"
    data-lightbox-items='[
        {"src":"photo1.jpg","alt":"Team 1"},
        {"src":"photo2.jpg","alt":"Team 2"}
    ]'
>View Team</button>

Omit data-lightbox-items if thumbnails with the same data-lightbox already exist.

5. For PDFs

<button data-lightbox-pdf="./path/to/document.pdf">View PDF</button>
<a href="#" data-lightbox-pdf="./path/to/document.pdf">Open Document</a>

Features

  • ✅ Display images in full-screen overlay
  • ✅ Display PDFs with dedicated full-height iframe layout
  • ✅ Thumbnail strip at bottom (auto-scrolls to active thumbnail)
  • ✅ Click thumbnails to jump to any page
  • ✅ Trigger galleries from buttons or links
  • ✅ Navigation buttons (prev/next)
  • ✅ Keyboard navigation (arrow keys, Escape)
  • ✅ Download button for PDFs
  • ✅ Close button and backdrop click to close
  • ✅ Page counter (e.g., "2 / 11")
  • ✅ Buttons hidden on first/last items
  • ✅ Active thumbnail highlighted
  • ✅ Fully responsive
  • ✅ Smooth animations
